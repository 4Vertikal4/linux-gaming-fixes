# 🎨 Warsztat GIMP (01_WORKSPACE_GIMP)

Ten folder służy WYŁĄCZNIE jako tymczasowe miejsce pracy nad grafiką.

**Zasady:**
1. Trzymamy tu pliki `.xcf` (projekty GIMP z warstwami).
2. Trzymamy tu wyciągnięte surowe `.dds` przed ich obróbką.
3. Po zakończeniu edycji, wyeksportowany, FINALNY plik `.dds` ląduje w `../02_READY_TO_INJECT`.
4. Po udanej iniekcji i teście, pliki z tego folderu przenosimy do `../_ARCHIVE_OLD`, aby utrzymać czystość dla kolejnych operacji.
