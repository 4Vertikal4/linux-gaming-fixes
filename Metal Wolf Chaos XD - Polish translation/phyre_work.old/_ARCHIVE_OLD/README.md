# 🏛️ Archiwum Projektu (Graveyard)

Ten folder zawiera stare wersje plików, nieudane eksperymenty oraz kopie zapasowe z poprzednich etapów prac.
**Nie używać tych plików do bieżącej iniekcji!**

## 📂 Struktura katalogów:

### `extracted_precision/`
*   **Zawartość:** Surowe pliki DDS wyciągnięte skryptem `15_Phyre_Precision_Extract` (Styczeń 2026).
*   **Status:** To są oryginalne ekstrakcje z gry, które potwierdziły rozdzielczość 1152x1152 (widać na nich "poszatkowane" napisy). Ważne jako punkt odniesienia.

### `Gotowe_menu_do_gry/`
*   **Zawartość:** Kopie zapasowe prac graficznych z GIMP-a (luty 2026).
*   **Pliki:** `menu_clean_stripes.dds` (wersja robocza), `menu_work.png`.

### `Stare_Proby_Iniekcji_Styczen/`
*   **Zawartość:** Pliki z nieudanych prób wstrzyknięcia (metody JSON/Mapowanie).
*   **Pliki:** `Inject_Payload...` oraz stare mapy `.json`. Te metody zawiodły.

### `modified/`
*   **Status:** Pusty folder, prawdopodobnie pozostałość po skrypcie `11` lub `12`. Do usunięcia.
